# BitRate – Lightweight Voice Assistant

**BitRate** is a simple Python-based virtual assistant that executes voice-activated system commands such as taking screenshots, opening applications, and displaying real-time feedback in a graphical interface. Designed with responsiveness and user experience in mind, BitRate leverages multithreading and a lightweight GUI to keep interactions smooth and intuitive.

---

## 🚀 Features
- 🎤 Voice recognition using the Google Speech API
- 🖼️ Take and save screenshots using PIL and OpenCV
- 🖥️ Open system applications through voice commands
- 🧠 Natural language processing with spaCy
- 🪟 Real-time GUI feedback via Tkinter
- 🔀 Multithreaded execution for responsive interaction

---

## 🛠️ Tech Stack
- Python 3.x
- [SpeechRecognition](https://pypi.org/project/SpeechRecognition/)
- [spaCy](https://spacy.io/)
- [Tkinter](https://docs.python.org/3/library/tkinter.html)
- [Pillow (PIL)](https://pillow.readthedocs.io/en/stable/)
- [OpenCV](https://opencv.org/)
- `threading` for concurrency
